Tabular ML Challenge

알고리즘은 자유입니다. 두 개의 예측 과제를 모두 제출하고, 평가셋 ROC AUC 평균으로 최종 순위를 정합니다.

공지 순위표 제출 데이터

대회 공지

두 데이터셋 합산 점수로 평가합니다

  • 모델은 자유 선택입니다. tree model, linear model, neural network, ensemble 모두 허용합니다.
  • 제출 파일은 dataset,id,prediction 세 컬럼을 포함해야 합니다.
  • 최종 점수는 고객 전환 예측 ROC AUC 50%와 신용 위험 예측 ROC AUC 50%의 평균입니다.
  • 팀별 제출은 한국 시간 기준 하루 3회까지만 채점합니다.
평가 지표 로딩 중
데이터셋 2종
제출 제한 하루 3회
업데이트 대기 중

Submission

제출 파일 형식

dataset,id,prediction
conversion,conversion_00101,0.7312
credit,credit_00108,0.1844

예측값은 0과 1 사이의 양성 클래스 확률이어야 합니다. 한 CSV 안에 두 데이터셋의 test row가 모두 들어가야 합니다.

운영 방식

자동 채점 흐름

  1. 팀이 hackerton/submissions/team.csv 파일로 PR을 생성합니다.
  2. 자동 채점기가 두 데이터셋 점수를 계산합니다.
  3. PR 댓글에 최종 점수와 데이터셋별 점수가 표시됩니다.
  4. 운영자가 PR을 merge하면 순위표가 자동으로 갱신됩니다.
