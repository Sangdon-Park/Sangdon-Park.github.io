Debugging Instructions

  1. Open Chrome DevTools (F12)
  2. Go to Network tab
  3. Filter by "gtm" or "analytics"
  4. Reload the page
  5. Check if GTM scripts are loading (should see gtm.js)
  6. Install Google Tag Assistant Chrome extension for detailed debugging
  7. Check Console for any JavaScript errors

Common Issues

  • Ad Blockers: Disable ad blockers as they may block Google Analytics
  • Container Not Published: Make sure GTM container is published in GTM interface
  • Wrong Container ID: Verify GTM-NKT8SV46 is correct
  • No GA4 Tag: Ensure GA4 tag is configured in GTM container
